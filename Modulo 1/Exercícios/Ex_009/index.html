Listas em html:

hoje iremos aprender a usar listas em html.

Listas Ordenadas:

Listas ordenadas são listas que se organizam usando numeradores como índice automaticamente, ou seja, é mais intuítivo usar esse tipo de lista, caso ocasionalmente você venha mudar algum ítem.

  1. Acordar.
  2. Tomar café.
  3. Comer tua mãe.
  4. Treinar calistenia.
  5. Programar.

DICA: apartir do html 5, é opicional fechar a tag < li >.

Lista Não Ordenadas

Listas não ordenadas são aquelas listas em que a sequencia não precisa fazer sentido.

  • planejamento;
  • mecanica;
  • level design;
  • direção de arte;
  • markenting;
  • lançamento;
